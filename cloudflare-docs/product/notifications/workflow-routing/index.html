Notifications / Workflow routing

Email me for assigned work and matching workflow changes.

Use Project > Settings > Notifications > Team workflow routing when the question is who should hear about assignment and status changes.

Settings

Routing is about matching the right work.

ControlMeaning
OffNo assignment or status-change emails.
Only errors assigned to meAssignment mail when the user becomes owner, and status-change mail for their assigned groups.
All matching workflow changesAny matching assignment or status change in the project.
Environment matchall or one exact environment such as production.
Severity matchall or one exact severity such as error.
Status matchUnresolved, Closed, or All.

Verify

Test with a real assignment.

  1. Make sure the recipient has project access.
  2. Assign an error group to the recipient.
  3. Change the group status as another user if you are testing assigned-to-me status mail.
