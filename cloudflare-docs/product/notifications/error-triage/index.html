Notifications / Error triage

Email me when one error group needs attention.

Use Project > Settings > Notifications > Error triage alerts when the alert should be about a single grouped error.

Settings

Each option answers a different question.

SettingWhat it means in the inboxDefault
New error groupsEmail when a brand-new grouped issue appears in the inbox.On
Reopened error groupsEmail when a group you had resolved, ignored, or archived gets a fresh occurrence and needs another look.On
Lifetime count milestonesEmail when one group's all-time occurrence count reaches a milestone, even if those events were spread out.Off
Single-error volume thresholdEmail when one grouped issue becomes noisy inside the recent window you choose.Off, 25 events in 60 minutes

Do not confuse the thresholds

Milestones answer "has this one issue become important over its whole life?" The single-error threshold answers "is this one issue suddenly noisy right now?" Project-wide spike alerts live on the Project health page because they count the whole project, not one group.

Verify

Trigger the exact condition you enabled.

  1. For new group mail, create a test error that has not appeared before.
  2. For reopened group mail, mark the group resolved or ignored, then send the same grouped error again.
  3. For frequent-error mail, temporarily lower the count and window, then send repeated events for the same grouped issue.

Troubleshooting

Check grouping before checking email delivery.

SymptomWhat to check
No new-group emailThe event may have grouped into an existing issue. Use a unique fingerprint or message for the test.
No reopened emailThe group may still be unresolved, or the new occurrence may not share the same grouping fingerprint.
No frequent-error emailThe repeated events may not have crossed the configured count inside the configured window.
The wrong person receives mailNotification preferences are per user and per project; check the recipient's own project settings.

Use notification setup requirements after the inbox state proves the alert condition happened.
