Fields

Each field controls a separate cleanup scope.

FieldWhat it controls
Hot event retentionNon-error ingest events such as logs, metrics, transactions, and check-ins that are not attached to an error group.
Trace span retentionSpan rows used for timing waterfalls and trace views.
Closed error retentionResolved, ignored, or archived error groups. Blank means keep closed groups indefinitely.
Enable archive exportsAllows retention runs to write archive records.
Archive before deleteRequires archive exports and writes a completed archive before matching rows are deleted.
